# TikFind 가격 전략 및 기능 분리

## 🎯 무료 미끼 서비스: TTS (Text-to-Speech)

### **전략:**
TTS를 무료로 제공하여 사용자가 TikFind의 가치를 경험하게 하고, 유료 기능으로 업그레이드 유도

---

## 📊 기능 분류

### ✅ **무료 기능 (Free Tier)**

#### 1. **TTS (Text-to-Speech)** 🔊
- 실시간 채팅 메시지 음성 읽기
- 다양한 음성 옵션
- 속도 조절
- 무제한 사용

**목적:** 
- 사용자가 TikFind의 핵심 가치를 경험
- 스트리밍 중 채팅을 놓치지 않게 도움
- 유료 전환 유도

#### 2. **기본 데이터 수집**
- 실시간 채팅 수집
- 시청자 수 표시
- 선물 알림
- 좋아요 카운트

#### 3. **웹 대시보드 접근**
- 실시간 통계 확인
- 기본 채팅 로그

---

### 💎 **유료 기능 (Pro/Premium Tier)**

#### 1. **AI 발음 코치** 🤖
- 외국어 자동 감지
- 실시간 번역
- 발음 가이드 (스트리머 언어로)
- AI 생성 답변 제안

**가치:**
- 외국인 시청자와 소통 가능
- 글로벌 팬 확보
- 전문적인 스트리밍

#### 2. **신청곡 시스템** 🎵
- `#노래제목#가수이름` 자동 파싱
- YouTube 자동 검색
- 우선순위 큐 관리
- 신청자 재실 확인

**가치:**
- 시청자 참여도 증가
- 방송 콘텐츠 풍부화
- 자동화된 관리

#### 3. **고급 분석** 📈
- 시청자 행동 분석
- 피크 타임 분석
- 인기 채팅 키워드
- 수익 최적화 제안

#### 4. **OBS 오버레이** 🎨
- 커스텀 디자인
- 실시간 채팅 표시
- 신청곡 표시
- 후원자 하이라이트

## 🎯 전환 전략

### **1단계: 무료 체험 (TTS)**
- 사용자가 TTS로 편리함 경험
- "채팅을 놓치지 않는다"는 가치 인식

### **2단계: 제한 노출**
- 외국어 채팅이 오면 "AI 발음 코치로 답변하세요!" 팝업
- 신청곡 형식 감지 시 "Pro로 업그레이드하면 자동 처리!" 알림

### **3단계: 가치 강조**
- "이번 달 외국인 시청자 234명과 소통 기회 놓침"
- "신청곡 45개 수동 처리 시간: 2시간 30분"

### **4단계: 쉬운 업그레이드**
- 클릭 한 번으로 업그레이드
- 7일 무료 체험 제공
- 언제든 취소 가능

---

## 📊 예상 전환율

### **무료 → Pro 전환율 목표: 5-10%**

**전환 촉진 요소:**
1. 외국인 시청자가 많은 스트리머
2. 음악 방송 스트리머
3. 성장 중인 스트리머 (100-1000 시청자)

**전환 방해 요소:**
1. 외국인 시청자가 거의 없음
2. 음악을 사용하지 않음
3. 매우 작은 채널 (<50 시청자)
