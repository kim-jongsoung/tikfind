## 🧪 테스트 시나리오

### **1단계: Desktop App 연결**

1. Desktop App이 실행되면 다음 정보 입력:
   - **User ID**: `6950e71049fd85ec7a001d6c`
   - **TikTok 사용자 이름**: `dodo_pilates` (@ 제외)

2. **"🚀 연결 시작"** 버튼 클릭

3. 연결 상태 확인:
   - 상태 표시등이 초록색으로 변경
   - "연결됨" 메시지 표시

---

### **2단계: TTS 무료 기능 테스트**

#### **확인 사항:**
- ✅ 실시간 채팅이 표시되는지
- ✅ 채팅 메시지가 음성으로 읽히는지 (TTS)
- ✅ 시청자 수가 업데이트되는지
- ✅ 선물이 표시되는지

#### **TTS 설정 (선택사항):**
- 음성 속도 조절
- 음성 선택
- 볼륨 조절

---

### **3단계: AI 발음 코치 테스트 (유료 기능)**

#### **현재 상태:**
- 테스트 User ID는 **trial 상태**이므로 AI 발음 코치 사용 가능

#### **테스트 방법:**
1. TikTok Live에서 외국어 채팅이 오면:
   - 영어: "Hello", "Hi", "Thanks"
   - 일본어: "こんにちは", "ありがとう"

2. Desktop App에서 확인:
   - **AI 발음 코치 섹션**에 발음 안내 표시
   - 원본 메시지
   - AI 답변
   - 발음 (한글로 표기)

#### **무료 사용자 테스트:**
- 구독 없는 User ID로 테스트하면:
  - AI 발음 코치 섹션에 "Pro로 업그레이드" 프롬프트 표시
  - 외국어 채팅이 와도 발음 안내 없음

---

### **4단계: 신청곡 시스템 테스트 (유료 기능)**

#### **신청곡 형식:**
```
#노래제목#가수이름
```

#### **테스트 예시:**
- `#Dynamite#BTS`
- `#롤린#브레이브걸스`
- `#Butter#BTS`

#### **확인 사항:**
1. 채팅에 신청곡 형식 입력
2. Desktop App의 **신청곡 큐 섹션** 확인:
   - 노래 제목, 가수 표시
   - YouTube 썸네일 (나중에 추가)
   - 신청자 이름
   - [▶️ 재생] [❌ 삭제] 버튼

---

## 🎯 무료 vs 유료 기능 확인

### **무료 기능 (항상 작동):**
- ✅ TTS (채팅 음성 읽기)
- ✅ 실시간 채팅 표시
- ✅ 시청자 수
- ✅ 선물 알림
- ✅ 기본 통계

### **유료 기능 (trial/active 상태만):**
- 💎 AI 발음 코치
- 💎 신청곡 자동 파싱
- 💎 YouTube 자동 검색

## 🔍 문제 해결

### **Desktop App이 실행 안 됨:**
```bash
cd c:\Users\kim\Desktop\tikfind\tikfind-desktop
npm install
npm start
```

### **서버 연결 실패:**
1. 서버가 실행 중인지 확인:
   ```bash
   cd c:\Users\kim\Desktop\tikfind
   npm start
   ```

2. 브라우저에서 확인:
   - http://localhost:3001

### **TTS가 작동 안 함:**
1. Windows 음성 설정 확인
2. 볼륨 확인
3. Desktop App 재시작

### **AI 발음 코치가 안 나옴:**
1. 구독 상태 확인 (trial 또는 active)
2. 외국어 채팅인지 확인 (한국어는 발음 코치 없음)
3. 서버 로그 확인
